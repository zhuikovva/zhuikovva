# Жуйков Всеволод - Аналитик данных
---
## Обо мне

Привет! Я — аналитик данных в фарм компании. Имею кандидатскую степень по биологии, но решил, что сфера аналитики интереснее, а главное прибыльнее. 
Мои основные должностные обязанности - поддержание актуальности данных о рынке БАДов, а также прогноз производства и продаж.

Основные достижение на данный момент:
1. Организую базу данных на базе postgreSQL с 0. Собираю из хаотичных данных, разбросанных по эксель-файлам и 1С в одну связанную, работающую БД.
2. Перевёл дашборд с рыночными данными из powerbi в datalens. Из не юзабельной системы получилось то, чем пользуются коллеги каждый день.
3. Автоматизирую прогнозные модели, правда пока только с использованием power query, но с прицелом на перевод на машинное обучение.
